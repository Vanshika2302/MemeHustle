#  MemeHustle

MemeHustle is a full-stack web app for creating, sharing, and viewing memes in real time. Built with React, Express, Supabase, and Socket.IO.

##  Tech Stack

- Frontend: **React**, **HTML**, **CSS**
- Backend: **Node.js**, **Express**
- Database: **Supabase**
